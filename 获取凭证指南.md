# 快速获取凭证指南

## 🔑 获取 Access Token（访问令牌）

### 步骤：

1. **打开Figma设置页面**
   - 访问：https://www.figma.com/settings
   - 或点击右上角头像 → Settings

2. **找到Personal access tokens**
   - 在左侧菜单选择 **Account**
   - 滚动到 **Personal access tokens** 部分

3. **创建新令牌**
   - 点击 **"Create new token"** 按钮
   - 输入名称（如：`figma-to-code`）
   - 点击 **"Create token"**

4. **复制令牌**
   - 令牌格式：`figd_xxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   - ⚠️ **重要**：令牌只显示一次，立即复制！
   - 粘贴到 `config.json` 的 `accessToken` 字段

---

## 📄 获取 File Key（文件ID）

### 步骤：

1. **打开你的Figma设计文件**
   - 在浏览器中打开要转换的Figma文件

2. **查看浏览器地址栏**
   - URL格式：`https://www.figma.com/file/XXXXXXXXXXXXXX/文件名`
   - 或：`https://www.figma.com/design/XXXXXXXXXXXXXX/文件名`

3. **复制File Key**
   - `XXXXXXXXXXXXXX` 就是File Key（在 `/file/` 或 `/design/` 之后）
   - 粘贴到 `config.json` 的 `fileKey` 字段

### 示例：

```
URL: https://www.figma.com/file/abc123def456ghi789/我的设计稿
File Key: abc123def456ghi789
```

---

## 🎯 获取 Node ID（节点ID，可选）

### 步骤：

1. **在Figma中选中元素**
   - 打开设计文件
   - 点击选中你想要转换的组件/框架/节点

2. **查看URL变化**
   - 地址栏会自动添加 `?node-id=XXX%3AYYY`
   - `XXX:YYY` 就是Node ID（`%3A` 是冒号的URL编码）

3. **复制Node ID**
   - 格式：`数字:数字`（如：`123:456`）
   - 粘贴到 `config.json` 的 `nodeId` 字段（可选）

### 示例：

```
URL: https://www.figma.com/file/abc123/设计?node-id=789%3A012
Node ID: 789:012
```

## 📝 完整配置示例

```json
{
  "figma": {
    "accessToken": "figd_abc123def456ghi789jkl012mno345pqr678",
    "fileKey": "abc123def456ghi789",
    "nodeId": "123:456"
  },
  "output": {
    "format": "react",
    "outputDir": "./output",
    "cssFramework": "none"
  }
}
```

## ❓ 常见问题

**Q: Token在哪里查看？**
A: 创建后只显示一次，如果忘记了需要删除旧token重新创建。

**Q: File Key找不到？**
A: 确保URL中包含 `/file/` 或 `/design/`，File Key就在这之后。

**Q: Node ID是必须的吗？**
A: 不是，如果不填 `nodeId`，会转换整个文件。

**Q: 可以转换团队文件吗？**
A: 可以，只要你有访问权限，File Key的获取方式相同。
